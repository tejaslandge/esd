1

Git Installation (Windows)

  • ➜ Download the Git installer from git-scm.com
  • ➜ Run the .exe file and click Next for all default options (recommended).
  • ➜ When asked for the default editor, you can choose VS Code or leave it as Vim.
  • ➜ Open Command Prompt or PowerShell and verify installation: 
git --version
2

GitHub Account Creation

Go to github.com/signup.

  • Enter your Email address.
  • Create a strong Password.
  • Choose a unique Username.
  • Verify your email via the code sent.

Ready to build the future!

3

Create a Repository

1

Click the + icon in the top right and select New repository.

2

Give it a name like my-first-project.

3

Keep it Public so others can see it.

4

Click Create repository.

4

Git Setup (First Time)

Only needs to be done once per computer.

Set Username

git config --global user.name "Your Name"

Set Email

git config --global user.email "your-email@example.com"
5

Clone Repository

Download your GitHub project to your computer:

git clone https://github.com/username/repository.git
cd repository
6

Basic Git Commands

Check Changes

git status

Add All Files

git add .

Save Changes (Commit)

git commit -m "My first commit"

Upload to GitHub

git push origin main
7

New Project Without Clone

Inside your local project folder:

git init
git add .
git commit -m "First commit"
git branch -M main
git remote add origin https://github.com/user/repo.git
git push -u origin main
8

GitHub Pages Deployment

Go to your repo and click Settings.

Click Pages in the left sidebar.

Under Build and deployment, select Main branch.

Click Save. Your URL will be live in 1-2 minutes!

URL format: https://username.github.io/repository/
